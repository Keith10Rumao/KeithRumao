### 🚀 Featured Projects

<details>
<summary><b>📚 StudyTrack — Study Management Platform</b></summary>
<br/>

A neo-brutalist study management web app with a live timer, assignment calendar, subject management, and Chart.js analytics — built and shipped end-to-end.

| | |
|---|---|
| **Stack** | Flask · SQLite · Gunicorn · Railway |
| **UI/UX** | Custom neo-brutalist dark theme, scroll-reveal animations, dot-grid & radial-glow visual system |
| **Auth & Security** | Werkzeug password hashing, session management, CSRF protection, account lockout |
| **Deployment** | Migrated to Railway with Gunicorn after diagnosing a data-loss issue caused by an ephemeral filesystem on the original hosting platform |
| **Documentation** | Professional README with badges and feature grid, published as part of a public developer portfolio |
| **Repository** | [github.com/Keith10Rumao/StudyTrack](https://github.com/Keith10Rumao/StudyTrack) |
| **Live Demo** | [web-production-43148.up.railway.app](https://web-production-43148.up.railway.app) |

Solved a real production incident: Vercel's ephemeral filesystem was silently dropping SQLite writes on every cold start. Diagnosed the root cause and re-architected the deployment on Railway with Gunicorn for persistent storage — a concrete debugging-under-pressure story, not just a shipped feature.

</details>
